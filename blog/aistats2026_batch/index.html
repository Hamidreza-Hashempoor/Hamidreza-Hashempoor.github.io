In this series I am collecting the accepted oral and spotlight papers from AISTATS 2026. For each paper I summarize the core idea, follow the key equations and theory, and add a small, self-contained experiment that reproduces the main qualitative claim — together with a companion notebook so the numbers can be re-run from scratch. The aim is not an exhaustive survey but a hands-on reading: enough theory to see why a method works, plus minimal code that makes the effect visible.

Each entry follows the same shape: idea → theory → minimal experiment → results → notebook. Experiments are deliberately tiny (often 1-D, NumPy-only) so they run in seconds and the theory stays in plain view.

  • Paper 1

    Synthetic-Regularized Kernel Regression

    Synthetic-data regularization • bias–variance • kernel methods

    A kernel regressor that trades noisy real labels against a synthetic generator $g$ via a penalty $\lambda\|f-g\|_{\mathcal H_K}^2$. Larger $\lambda$ cuts variance but adds bias when $g\neq f_\star$; the best $\lambda$ depends on how well $g$ matches the target — especially in the hard, high-frequency kernel directions. Verified on a 1-D RBF experiment.

  • Paper 2

    EventFlow — Flow Matching for Temporal Point Processes

    temporal point processes • flow matching • generative models

    A non-autoregressive generative model for event sequences. A balanced coupling pairs reference and data sequences of equal length, a linear noisy interpolant defines the flow, and a vector field trained by masked MSE transports a uniform reference TPP onto the data distribution — preserving the event count while pulling event times onto the data clusters. Verified on a clustered synthetic TPP.
